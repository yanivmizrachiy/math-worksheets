## 📈 Desmos - מחשבון גרפי

### מה זה?
מחשבון גרפי אינטראקטיבי חזק ליצירת גרפים, פתרון משוואות ועבודה מתמטית מתקדמת.

### קישורים:
- **מחשבון גרפי:** https://www.desmos.com/calculator
- **גיאומטריה:** https://www.desmos.com/geometry
- **מדעים:** https://www.desmos.com/scientific
- **מטריצות:** https://www.desmos.com/matrix

### שימושים:
1. **ציור גרפים** - פונקציות, משוואות, אי-שוויונים
2. **טבלאות** - יצירת טבלאות נתונים
3. **רגרסיות** - התאמת עקומות
4. **אנימציות** - הדמיות דינמיות
5. **סטטיסטיקה** - גרפים וניתוחים

### דוגמאות שימוש:

#### גרף פונקציה ליניארית:
```
y = 2x + 1
```

#### גרף פונקציה ריבועית:
```
y = x^2 - 4x + 3
```

#### גרף טריגונומטרי:
```
y = sin(x)
```

## 🔷 GeoGebra - מתמטיקה דינמית

### מה זה?
כלי מתמטיקה דינמי המשלב גיאומטריה, אלגברה, סטטיסטיקה וחישוב.

### קישורים:
- **מחשבון:** https://www.geogebra.org/calculator
- **גיאומטריה:** https://www.geogebra.org/geometry
- **3D:** https://www.geogebra.org/3d
- **CAS:** https://www.geogebra.org/cas

### שימושים:
1. **גיאומטריה** - בניית צורות, מדידות, הוכחות
2. **אלגברה** - פתרון משוואות, גרפים
3. **סטטיסטיקה** - גרפים, הסתברות
4. **חשבון** - נגזרות, אינטגרלים
5. **3D** - גיאומטריה תלת-ממדית

### דוגמאות שימוש:

#### בניית משולש:
1. פתח GeoGebra Geometry
2. בחר "Polygon"
3. צייר 3 נקודות
4. המשולש יווצר אוטומטית

#### גרף פונקציה:
1. פתח GeoGebra Calculator
2. הקלד: `f(x) = x^2`
3. הגרף יוצג אוטומטית

## 💡 טיפים לשימוש

### Desmos:
- ✅ השתמש ב-`y = ...` לגרפים
- ✅ השתמש ב-`y_1 = ...` ו-`y_2 = ...` להשוואות
- ✅ השתמש ב-table mode לטבלאות
- ✅ השתמש ב-sliders לאנימציות

### GeoGebra:
- ✅ שמור את העבודה שלך (Sign In)
- ✅ השתמש ב-Share ליצירת קישור
- ✅ השתמש ב-Export ליצירת תמונה
- ✅ השתמש ב-Construction Protocol לראות שלבים

## 🎓 דוגמאות מעשיות
