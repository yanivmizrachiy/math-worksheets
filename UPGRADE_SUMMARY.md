## 🎯 איך להשתמש בכלים החדשים?

### VS Code Extensions:
1. פתח VS Code
2. לחץ `Ctrl+Shift+X` (Extensions)
3. לחץ על "Install Recommended Extensions"

### npm Scripts:
```bash
npm install              # התקן dependencies
npm run build            # בנה PDF
npm run build:watch      # build אוטומטי
npm run sync             # סנכרן ל-GitHub
npm run serve            # הפעל שרת מקומי
```

### GitHub Actions:
- רץ אוטומטית אחרי כל push
- PDF נשמר כ-Artifact
