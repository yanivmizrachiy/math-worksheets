# הגדרת קישור לשולחן העבודה

**קישור קבוע לאתר index.html - נשמר ומסונכרן אוטומטית!**

---

## 🎯 מה זה עושה?

יוצר קישור לשולחן העבודה שמפנה ישירות לאתר `index.html` שלך.

**יתרונות:**
- ✅ קישור קבוע לשולחן העבודה
- ✅ נשמר בפרויקט ונסנכרן ל-GitHub
- ✅ נשאר גם אחרי עדכונים
- ✅ פתיחה מהירה עם לחיצה כפולה

---

## 🚀 איך להשתמש?

### אופציה 1: הרצה ישירה
```powershell
cd C:\Users\yaniv\math-worksheets
.\CREATE_DESKTOP_LINK.ps1
```

### אופציה 2: דרך VS Code
1. לחץ `Ctrl+Shift+P`
2. הקלד "Run Task"
3. בחר "Create Desktop Link"

### אופציה 3: קיצור דרך npm
```bash
npm run desktop-link
```

---

## 📋 מה קורה?

1. **יוצר קישור לשולחן העבודה** - `מערכת דפי עבודה במתמטיקה.url`
2. **יוצר קישור בפרויקט** - נשמר בקובץ `.url` בפרויקט
3. **מוסיף ל-Git** - מוכן לסנכרון
4. **נשמר** - גם אחרי עדכונים ועדכוני Git

---

## 🔄 סנכרון אוטומטי

הקישור יסונכרן אוטומטית ל-GitHub:
- אחרי `git commit`
- דרך `AUTO_SYNC_GITHUB.ps1`
- דרך `SYNC_TO_GITHUB.ps1`

---

## 🛠️ עדכון הקישור

אם שינית את מיקום הפרויקט או את שם הקובץ:
```powershell
.\CREATE_DESKTOP_LINK.ps1
```

הקישור יתעדכן אוטומטית!
