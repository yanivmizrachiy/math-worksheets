# 📚 מפעל דפי עבודה במתמטיקה - LaTeX

## מה זה?
מערכת מלאה ליצירת דפי עבודה מתמטיים בעברית עם XeLaTeX.

### מה כולל:
- ✅ עברית RTL תקינה
- ✅ גרפים וקטוריים (ליניארי/ריבועי)
- ✅ ציורים גיאומטריים (TikZ)
- ✅ טבלאות ותבניות מותאמות
- ✅ 7 סניפטים ל-VS Code
- ✅ בניית PDF אוטומטית

## 📂 קבצים זמינים

כל הקבצים זמינים בשיחת ה-Comet שיצרה אותם!

📄 **קבצים להורדה:**
1. `main.tex` - מסמך ראשי (~150 שורות)
2. `worksheet.sty` - סגנון (~200 שורות)
3. `settings.json` - VS Code (~30 שורות)
4. `latex.json` - סניפטים (~80 שורות)

## 🛠️ התקנה מהירה

### 1. הורד כלים:
- [VS Code](https://code.visualstudio.com/download)
- [MiKTeX](https://miktex.org/download)

### 2. התקן תוסף:
ב-VS Code: Ctrl+Shift+X → חפש "LaTeX Workshop" → Install

### 3. צור תיקיות:
```
math-worksheets/
├── tex/
├── out/
└── .vscode/
```

### 4. העתק קבצים:
העתק את 4 הקבצים מהשיחה למיקומות המתאימים

### 5. בנה PDF:
פתח `main.tex` ב-VS Code → Ctrl+S → ה-PDF יבנה אוטומטית!

## 🎯 שימוש

השתמש בסניפטים:
- `qst` → שאלה
- `linplot` → גרף קווי
- `quadplot` → גרף ריבועי
- `numline` → ציר מספרים
- `tbl` → טבלה
